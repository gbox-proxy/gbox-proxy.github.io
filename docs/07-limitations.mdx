Currently, the GBox have some limitations below.

+ Upstream **must** accept GBox request an [introspection query](https://spec.graphql.org/June2018/#sec-Introspection) to known upstream schema.
+ Only support GraphQL requests over HTTP `POST` method.
+ Only support GraphQL subscription by the following transports:
  - [graphql-ws](https://github.com/enisdenjo/graphql-ws)
  - [subscriptions-transport-ws](https://github.com/apollographql/subscriptions-transport-ws)
